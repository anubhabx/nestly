# @specord/types

Shared TypeScript contracts for Specord packages.

This package contains the V1 inspection model, diagnostics, and configuration types used by the extractor, CLI, OpenAPI emitter, and NestJS adapter.

## Install

```bash
npm install @specord/types
```

## Status

Specord V1 is NestJS-first and source-analysis based. See the root repository for the current roadmap and supported behavior.
